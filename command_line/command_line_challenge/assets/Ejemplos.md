### Ejemplo 1

- mkdir vamos a crear dos carpetas

```bash
mkdir prueba
mkdir -p prueba/contenido1
```

### Ejemplo 2

- cp copiaremos el archivo prueba.txt a un nuevo archivo prueba2.txt

```bash
cp prueba.txt prueba2.txt
```

### Ejemplo 3

- mv vamos a mover el archivo a la ruta especificada

```bash
mv prueba.txt /home/user/main/
```
