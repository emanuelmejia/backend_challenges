# Command Line o Linea de Comandos

la linea de comandos es una interfaz de usuario que permite a los usuarios interactuar con el sistema operativo o software mediante la introducción de comandos de texto. A diferencia de las interfaces gráficas de usuario (GUI), donde los usuarios hacen clic en iconos y menús, la línea de comandos requiere que los usuarios escriban comandos específicos para realizar tareas. 
Algunos ejemplos comunes de líneas de comandos incluyen la Terminal en macOS y Linux, y el Símbolo del sistema (Command Prompt) o PowerShell en Windows.

Las líneas de comandos son especialmente útiles para tareas avanzadas, automatización, administración del sistema y desarrollo de software, ya que permiten un control más preciso y eficiente sobre las operaciones del sistema.

Ejemplos de comandos comunes en la línea de comandos incluyen:
- cd (change directory): Cambia el directorio actual.
- ls (list): Lista los archivos y carpetas en el directorio actual (en Windows, se usa dir).
- mkdir (make directory): Crea un nuevo directorio.
- rm (remove): Elimina archivos o directorios.
- cp (copy): Copia archivos o directorios.
- mv (move): Mueve o renombra archivos o directorios.
- echo: Muestra un mensaje en la pantalla.
- ping: Verifica la conectividad de red con otro dispositivo.
- nano: editor de texto en la terminal (macos)

Las líneas de comandos pueden parecer dificil al comienzo, pero prácticando, pueden convertirse en una herramienta poderosa para usuarios avanzados y desarrolladores.
